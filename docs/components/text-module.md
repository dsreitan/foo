# Text Module

Colored text section for long-form content (brand/neutral × tone a/b).

## Built-in behavior

- `<section>`; text color cascades from the surface (tone-a = light text)

## Usage responsibilities

- Give the section an accessible name when it's a landmark:
  `aria-label` or `aria-labelledby` pointing at its heading.
- Keep heading levels inside consistent with the page outline.
